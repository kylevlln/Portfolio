# CHOC_CORE // Personal Portfolio

A cinematic, cyberbrutalist, and terminal-inspired portfolio website created by Spencer. Built entirely with pure **HTML**, custom **CSS**, and **Vanilla JavaScript**, this site is designed to showcase projects, technical telemetry, and capabilities with a high-end, intentional aesthetic. 

## 🚀 Features

* **⚡ [00] INITIATOR (Hero):** Striking monumental typography, abstract wireframe visuals, and smooth native scroll animations.
* **🗂️ [01] TELEMETRY (About):** An asymmetrical bento grid detailing biometric records, operational capacity, and personal notes.
* **📂 [02] ARCHIVES (Projects):** A dynamic project registry featuring real-time custom cursor image previews. Includes core projects like *QueueCare* and *ReVocab*.
* **🛠️ [03] STACK (Tech Stack):** An interactive, clickable technology matrix showcasing frontend and backend capabilities.
* **📡 [04] UPLINK (Contact):** Handshake initiation (contact links) and live system telemetry, including a temporal clock synced to Philippine Standard Time (PST).
* **🤖 CHOC_AI Terminal:** A built-in, retro-futuristic command-line interface chatbox with custom keyword recognition to answer guest queries.

## 🛠️ Built With

* **HTML5:** Semantic architecture and custom data attributes.
* **CSS3:** Native smooth scrolling, CSS variables, custom cursors, scanline overlays, and `IntersectionObserver` fade-in animations.
* **JavaScript:** Dynamic project rendering, custom mouse tracking, live clock synchronization, and a lightweight keyword-based AI chat engine.
